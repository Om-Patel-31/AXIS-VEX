System Overview

AXIS is a comprehensive robotics control library designed for VEX V5 competition systems. It provides three integrated modules that work together to deliver responsive driver control, accurate positioning, and precise automated movement.

Core Features

  • DriverControl: Responsive input processing with multiple control paradigms, customizable exponential curves, and joystick sensitivity calibration. Supports arcade, tank, split arcade, curvature, and single-stick modes.
  • OdometryTracker: Real-time position and orientation estimation using dual encoder feedback and IMU fusion. Maintains field-centric tracking with drift correction and supports both relative and absolute positioning modes.
  • PIDController: Industrial-grade proportional-integral-derivative controller with integral windup protection, configurable tolerance thresholds, and comprehensive tuning utilities. Ready for distance, heading, and multi-axis control applications.

Why AXIS

  • Production-ready code with extensive testing and optimization
  • Flexible architecture supports multiple drivetrain configurations and control schemes
  • Comprehensive documentation with practical implementation examples
  • Performance-optimized for real-time competition environments
  • Modular design allows independent use of any component

Adjust these properties at runtime to fine-tune controller behavior:

lastI double 0.0 Previous integral component

API Methods

calculate()

double calculate(double error, double dt)

Computes PID control output based on error feedback.

  • error: Current error value (target - actual)
  • dt: Time delta from previous calculation (seconds)
  • Returns: Control signal output

reset()

void reset()

Clears integral accumulation and error history.

atTarget()

bool atTarget(double error)

Convergence detection within tolerance bounds.

Tuning Guidelines

Follow this systematic process to achieve optimal controller performance:

  • Step 1 (kP): Start with 0.5 and gradually increase until the system oscillates. Back off to 70% of that value.
  • Step 2 (kD): Increase from 0.1 until oscillation is eliminated. This parameter damps overshoot.
  • Step 3 (kI): Increase from 0.01 until steady-state error is eliminated. Use sparingly to avoid instability.
  • Validation: Test at various setpoints to confirm stability across the operating range.

Distance Control Example

Autonomous drive-to-distance routine using PID control:

PIDController drivePID(1.0, 0.0, 0.1);
drivePID.tolerance = 0.5;  // 0.5 inch precision

double target = 24.0;  // 24 inch displacement
double lastTime = Brain.timer(msec);

while (!drivePID.atTarget(error)) {
    double current = leftEncoder.position(inches);
    double error = target - current;
    
    double currentTime = Brain.timer(msec);
    double dt = (currentTime - lastTime) / 1000.0;
    lastTime = currentTime;
    
    double power = drivePID.calculate(error, dt);
    
    leftMotor.spin(forward, power, percent);
    rightMotor.spin(forward, power, percent);
    
    wait(20, msec);
}

}

Complete System Integration

Full project setup demonstrating all three modules working together:

#include "vex.h"
#include "DriverControl.h"
#include "OdometryTracker.h"
#include "PIDController.h"

using namespace vex;

competition Competition;

// Hardware configuration
controller Controller1 = controller(primary);
motor leftFront = motor(PORT1);
motor leftBack = motor(PORT2);
motor rightFront = motor(PORT3);
motor rightBack = motor(PORT4);
inertial imu = inertial(PORT10);

// Neural module instances
DriverControl driverControl(ControlType::ARCADE, 10, 2.0, 60, 50);
OdometryTracker odometry;
PIDController drivePID(1.0, 0.0, 0.1);
PIDController turnPID(1.5, 0.0, 0.2);

void pre_auton(void) {
  // Motor configuration
  leftFront.setBrake(brake);
  leftBack.setBrake(brake);
  rightFront.setBrake(brake);
  rightBack.setBrake(brake);
  
  // IMU calibration sequence
  imu.calibrate();
  waitUntil(!imu.isCalibrating());
  
  // Odometry initialization
  odometry.reset();
}

void autonomous(void) {
  // Autonomous control matrix
}

void usercontrol(void) {
  while (true) {
    double leftPower, rightPower;
    driverControl.calculate(Controller1, leftPower, rightPower);
    
    leftFront.spin(forward, leftPower, percent);
    leftBack.spin(forward, leftPower, percent);
    rightFront.spin(forward, rightPower, percent);
    rightBack.spin(forward, rightPower, percent);
    
    wait(20, msec);
  }
}

int main() {
  Competition.autonomous(autonomous);
  Competition.drivercontrol(usercontrol);
  
  pre_auton();
  
  while (true) {
    wait(100, msec);
  }
}

Advanced: Autonomous Navigation

Field-centric navigation combining odometry tracking with dual PID controllers for simultaneous distance and heading control:

void driveToPoint(double targetX, double targetY) {
    PIDController distPID(1.0, 0.0, 0.1);
    PIDController headingPID(1.5, 0.0, 0.2);
    
    distPID.tolerance = 1.0;
    headingPID.tolerance = 2.0;
    
    double lastTime = Brain.timer(msec);
    
    while (true) {
        // Odometry update cycle
        double leftDist = (leftFront.position(degrees) + leftBack.position(degrees)) / 2.0;
        double rightDist = (rightFront.position(degrees) + rightBack.position(degrees)) / 2.0;
        odometry.update(leftDist * 0.0872665, rightDist * 0.0872665, imu.heading());
        
        // Error calculation
        double distError = odometry.distanceTo(targetX, targetY);
        double targetHeading = odometry.headingTo(targetX, targetY);
        double headingError = targetHeading - odometry.heading;
        
        // Angular wraparound normalization
        if (headingError > 180) headingError -= 360;
        if (headingError < -180) headingError += 360;
        
        // Convergence detection
        if (distPID.atTarget(distError) && headingPID.atTarget(headingError)) {
            break;
        }
        
        // Time delta computation
        double currentTime = Brain.timer(msec);
        double dt = (currentTime - lastTime) / 1000.0;
        lastTime = currentTime;
        
        // PID control signals
        double distPower = distPID.calculate(distError, dt);
        double turnPower = headingPID.calculate(headingError, dt);
        
        // Differential drive matrix
        double leftPower = distPower + turnPower;
        double rightPower = distPower - turnPower;
        
        leftFront.spin(forward, leftPower, percent);
        leftBack.spin(forward, leftPower, percent);
        rightFront.spin(forward, rightPower, percent);
        rightBack.spin(forward, rightPower, percent);
        
        wait(20, msec);
    }
    
    // Termination sequence
    leftFront.stop();
    leftBack.stop();
    rightFront.stop();
    rightBack.stop();
}

Proportional-Integral-Derivative controller for precise automated movements.

Properties

Property Type Default Description

lastP double 0.0 Last proportional component
lastI double 0.0 Last integral component
lastD double 0.0 Last derivative component

Methods

calculate()

double calculate(double error, double dt)

Calculates PID output based on current error.

  • error: Current error value (target - current)
  • dt: Time delta since last calculation (seconds)
  • Returns: Control output value

reset()

void reset()

Resets integral accumulation and previous error.

atTarget()

bool atTarget(double error)

Checks if error is within tolerance.

PID Tuning Tips:
  • kP: Start with small value (0.5-2.0). Increase until oscillation occurs.
  • kI: Usually very small (0.0-0.1). Helps eliminate steady-state error.
  • kD: Reduces oscillation. Start small (0.1-0.5).
  • Tune one gain at a time: P first, then D, then I.

void driveToPoint(double targetX, double targetY) {
    PIDController distPID(1.0, 0.0, 0.1);
    PIDController headingPID(1.5, 0.0, 0.2);
    
    distPID.tolerance = 1.0;
    headingPID.tolerance = 2.0;
    
    double lastTime = Brain.timer(msec);
    
    while (true) {
        // Update odometry
        double leftDist = (leftFront.position(degrees) + leftBack.position(degrees)) / 2.0;
        double rightDist = (rightFront.position(degrees) + rightBack.position(degrees)) / 2.0;
        odometry.update(leftDist * 0.0872665, rightDist * 0.0872665, imu.heading());
        
        // Calculate errors
        double distError = odometry.distanceTo(targetX, targetY);
        double targetHeading = odometry.headingTo(targetX, targetY);
        double headingError = targetHeading - odometry.heading;
        
        // Handle angle wraparound
        if (headingError > 180) headingError -= 360;
        if (headingError < -180) headingError += 360;
        
        // Check if at target
        if (distPID.atTarget(distError) && headingPID.atTarget(headingError)) {
            break;
        }
        
        // Calculate dt
        double currentTime = Brain.timer(msec);
        double dt = (currentTime - lastTime) / 1000.0;
        lastTime = currentTime;
        
        // Calculate PID outputs
        double distPower = distPID.calculate(distError, dt);
        double turnPower = headingPID.calculate(headingError, dt);
        
        // Apply to motors
        double leftPower = distPower + turnPower;
        double rightPower = distPower - turnPower;
        
        leftFront.spin(forward, leftPower, percent);
        leftBack.spin(forward, leftPower, percent);
        rightFront.spin(forward, rightPower, percent);
        rightBack.spin(forward, rightPower, percent);
        
        wait(20, msec);
    }
    
    // Stop motors
    leftFront.stop();
    leftBack.stop();
    rightFront.stop();
    rightBack.stop();
}
